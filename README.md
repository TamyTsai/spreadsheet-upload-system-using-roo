# 使用者試算表上傳系統

## 關於使用者試算表上傳系統
- 一個可以將Excel檔案資料直接匯入資料表，然後呈現出來的網頁
  
<!-- ## 專案畫面與功能介紹 -->

## 安裝
以下皆為於macOS環境運行
### 安裝Ruby
```bash
$ brew install ruby
```
### 安裝Rails v7.1.3.4
```bash
$ gem install rails -v 7.1.3.4
```
### 安裝相關套件
```bash
$ bundle install
```
### 啟動rails伺服器及編譯器
```bash
$ rails s
```
### 開啟專案
在瀏覽器網址列輸入以下網址即可看到專案首頁
```bash
http://localhost:3000/
```

## 專案技術
- 前端：HTML、Simple.css
- 後端：Ruby on Rails
- 版本控制：Git

## 使用技術詳細說明
- 以HTML、CSS及Ruby撰寫，為動態網頁
- 使用Simple.css做簡單快速的網頁美化
- 使用Rails框架製作網頁
- 使用Roo套件，讓使用者可以使用Excel匯入資料
